DoramaSeeker
===

为了方便下载日剧，写了这个小脚本。

用法
--

**第一步、代码克隆到本地**

```bash
git clone git@github.com:LyndonChin/everjapan-dorama.git
nmp install
```

**第二步、从下载页面中解析出下载链接**

```bash
node seeker.js http://xxx.com/xxx -p ed2k -t tab-g101-HR-HDTV
```

**第三步、将解析出来的链接导入到下载工具**
